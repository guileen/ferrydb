# FerryDB

Fast databse base on [rocksdb](http://rocksdb.org).

# Features

* Rocksdb.
* Easy to install.
* Compress with LZ4.

# Install

Download the source code.

```
make
```

# Note

Mac OSX is not support rocksdb compile with lz4.
